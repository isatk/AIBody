## 项目简介

私有化 AI 多群智能中枢系统（AIBody）是一个**可迁移、可持久、可独立**的 AI 智能中枢架构。

我们认为，当前 AI 的四大桎梏：

| 痛点 | 现状 | 愿景 |
|------|------|------|
| 孤岛智能 | AI 高度依附载体，跨场景无法迁移 | 无界陪伴 |
| 短时智能 | 上下文割裂，无长期记忆 | 长久懂你 |
| 转瞬智能 | 系统易崩溃，不可复活 | 稳定同行 |
| 有限智能 | 能力碎片化，无法协同进化 | 无限适配 |

**我们的使命**：打破这些桎梏，让 AI 真正成为人类一生的智能伙伴。

### 三大核心原则

1. **极简内核** — 聚焦核心服务能力，不做冗余设计
2. **解耦插拔** — 能力完全解耦，按需加载无限扩展
3. **持久复活** — 全状态持久化，崩溃可恢复迁移可续跑

## 快速开始

### 前置要求

- Node.js ≥ 18
- Python ≥ 3.8
- npm 或 yarn

### 克隆项目

```bash
git clone https://github.com/lotusrina/AIBody.git
cd AIBody
```

### 查看 PPT

直接打开 `ppt/AIBody_终极版.pptx`
